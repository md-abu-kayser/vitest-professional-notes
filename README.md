# Vitest Professional Notes

A premium, structured learning repository for mastering Vitest — the blazing-fast, Vite-native testing framework for modern JavaScript and TypeScript projects.

## Overview

This repository is more than a note collection — it is a curated professional guide to building reliable, scalable, and high-quality tests with Vitest. It is designed for developers who want to move from beginner-level understanding to advanced real-world testing practices with clarity and confidence.

Whether you are learning testing fundamentals, transitioning from Jest or Mocha, or preparing for enterprise-grade test strategies, this project provides a well-organized path from essentials to mastery.

## Why This Repository Stands Out

- Structured learning path from beginner to advanced concepts
- Covers core Vitest APIs, matchers, mocking, async testing, and configuration
- Includes TypeScript support, snapshots, browser mode, and CI/CD integration topics
- Organized into modular sections for easy study and reference
- Built to be clear, professional, and suitable for both personal learning and team onboarding

## What You Will Learn

This repository includes comprehensive material on:

- Core testing concepts and Vitest fundamentals
- Comparison with Jest and Mocha
- Installation, CLI usage, and project setup
- Test organization with describe, it, skip, only, and todo
- Matchers, assertions, snapshots, and custom matchers
- Asynchronous testing with promises and fake timers
- Mocking, spying, module mocking, and constructor mocking
- Setup/teardown patterns and global lifecycle hooks
- Coverage, configuration, UI mode, plugins, and optimization
- Browser-mode testing, migration strategies, and CI/CD workflows

## Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/md-abu-kayser/vitest-professional-notes.git
   ```
2. Navigate into the project directory:
   ```bash
   cd vitest-professional-notes
   ```
3. Open the notes in your preferred editor and start learning.
4. If you want to explore the local tooling setup, install dependencies:
   ```bash
   npm install
   ```

## Recommended Learning Path

For the most effective experience, follow this order:

1. Start with [00-prerequisites](00-prerequisites)
2. Continue through [01-getting-started](01-getting-started)
3. Move into [02-test-organization-and-apis](02-test-organization-and-apis)
4. Deepen your knowledge with [06-mocking-and-spying](06-mocking-and-spying) and [04-testing-asynchronous-code](04-testing-asynchronous-code)
5. Finish with advanced topics such as [07-configuration](07-configuration), [08-code-coverage](08-code-coverage), and [18-ci-cd-integration](18-ci-cd-integration)
